## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Hairline visual offsets, tiny icon adjustments |
| sm | 8px | Tight icon-text gaps and nav micro spacing |
| md | 16px | Default inline navbar spacing |
| lg | 24px | Internal dock breathing room |
| xl | 32px | Desktop nav cluster spacing cadence |
| 2xl | 48px | Major left/center/right zone separation |
| 3xl | 64px | Top visual rhythm on very wide viewports |

Exceptions:
- Touch targets: all interactive nav controls stay >=44px (`min-h-11` / `min-w-11`).
- Fixed header offset remains `scroll-padding-top: 5.5rem` to prevent anchor clipping.

|------|------|--------|-------------|
| Body | 15px | 500 | 1.4 |
| Label | 13px | 600 | 1.3 |
| Heading | 16px (brand wordmark) | 600 | 1.25 |
| Display | 15px (nav items) | 600 | 1.3 |

Typography rules:
- Primary nav labels use `text-[15px] font-semibold` only.
- Do not mix serif display styling into nav links; keep product SaaS tone.
- Conference emphasis can increase tracking slightly (`tracking-[0.01em]`) but not size inflation.

## Navbar Visual Direction (Locked)

### Surface and chrome
- Keep floating dock architecture (`nav-dock nav-dock--expandable`) with restrained frosted glass.
- Reduce visual noise: one soft outer shadow + one inner hairline only; avoid stacked dramatic glows.
- Border should remain subtle (`~6-8%` black alpha in light mode), increased slightly on hover/open state.

### Layout and order
- Locked desktop/mobile nav order: `Blog`, `Events`, `Conference`, `Community`, `About Us`.
- Keep 3-zone desktop grid (`brand | nav | CTA`) already present in `nav-dock__bar`.
- Maintain max width (`max-w-6xl`) for premium but calm density.

### Link styling
- Base links: neutral text tint with rounded-full hover chip.
- Hover: only color + background transitions (`transition-colors`), 180-220ms.
- Active route: same chip language as hover, persistent with slightly stronger text contrast.

### Conference emphasis (elegant, not loud)
- Keep Conference as a filled premium chip using inverse contrast:
  - light routes: `bg-text text-white border border-text`.
  - conference route (dark context): `bg-white text-black border border-white`.
- No pulsing, no gradients, no glow animation.
- Hover delta should be subtle opacity shift (`/90` or `/95`), not scale bounce.

---

## Interaction Contract

### Desktop
- Hover/focus behavior uses color/opacity only; no layout shift and no width animation.
- `focus-visible` rings remain on logo, links, CTA, and menu trigger with consistent ring token.
- Tab order: brand -> nav items in visual order -> primary CTA -> mobile trigger (mobile only).

### Mobile menu
- Trigger retains current icon crossfade but with transform+opacity only.
- Drawer open/close continues with `grid-template-rows` + opacity; no JS animation library.
- Close behaviors required: backdrop click, Escape key, route change.
- Body should remain scroll-locked while menu is open (if not already enforced, add class toggle).
- Link rows keep `min-h-[3.25rem]` and full-width tap area.

### Reduced motion
- Existing `prefers-reduced-motion` block remains mandatory: no icon transitions, no drawer fade timing.
- Mobile open/close should become near-instant with motion reduction enabled.
